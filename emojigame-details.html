Project Details

During my course on cloud computing at Generation Australia, I developed a game called "Emoji Guess Game" using Python.
This game showcases my fundamental Python skills that I acquired as part of the program.

The "Emoji Guess Game" allows users to select a category (countries or food) and guess the corresponding word based on the displayed emoji. Each guess has a time limit of 20 seconds, and the score is calculated based on the number of correct answers. The game can be played multiple times.

This project demonstrates my ability to work with Python, utilize external modules, handle user input, implement timers, and create interactive command-line applications.
